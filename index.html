安装指南

下载安装程序

点击页面顶部的"下载安装程序"按钮，将安装文件保存到您的计算机

安装程序下载示意图

运行安装向导

找到下载的安装文件（通常位于"下载"文件夹），双击运行

如果系统提示安全警告，请选择"运行"或"更多信息">"仍要运行"

安全警告示意图

完成安装

按照安装向导的提示进行操作：

  1. 接受许可协议
  2. 选择安装位置（推荐使用默认路径）
  3. 选择开始菜单文件夹
  4. 点击"安装"按钮开始安装
安装向导示意图

启动应用程序

安装完成后，您可以选择立即启动应用程序

您可以通过以下方式访问应用程序：

  • 桌面快捷方式
  • 开始菜单 > TuSirSOFT
  • 安装目录中的可执行文件

系统要求

  • 操作系统 Windows 10/11, macOS 12+, Linux Ubuntu 20.04+
  • 处理器 1.5 GHz 双核或更高
  • 内存 4 GB RAM (推荐 8 GB)
  • 存储空间 安装需要 500 MB 可用空间
  • Java环境 Java Runtime Environment (JRE) 11 或更高版本
  • 显示器 1024×768 分辨率或更高
